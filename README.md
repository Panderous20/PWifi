# PWifi

> **Note**: This app is my final year project, so many features are still under construction and will be improved in the future.

An open source Android app for monitoring Wi-Fi networks.  
It supports 2.4/5/6 GHz Wi-Fi and uses a LibreSpeed server to measure download and upload speeds.


## Features

- Scan available networks
- Show signal strength (RSSI)
- Measure ping and jitter
- Download speed test
- Upload speed test
- Chart RSSI for 30 seconds
- Export results to file
